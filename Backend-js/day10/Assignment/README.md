# Assignment

### index.js에서 `pencil` 함수를 만들어주세요.

`pencil` 함수는 학생 1인당 연필을 1자루씩 나누어 준다고 할 때 필요한 연필의 다스 수를 반환해야 합니다.

- `pencil`이라는 이름의 함수를 만들어주세요.
- 임의의 숫자를 인자로 받으며 학생 수를 의미합니다.
- 연필 1 다스는 12자루입니다.
- 인자 이름은 원하는 대로 지어주셔도 됩니다.

**참고사항**
- `Math.ceil()`에 대해서 공부해 보시고 적용해 보세요.

**입력예제1**
- 25

**출력예제1**
- 3

**입력예제2**
- 178

**출력예제2**
- 15
